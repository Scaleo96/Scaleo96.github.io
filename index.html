A NEAT Project

This is a project which implemented NeuroEvolution of Augmenting Topologies (NEAT) with the goal to learn how to play Super Mario. Input to the neural network was accomplished by a grid that 'visualized' the screen, each box in the grid corresponded to an input node. When an entity intersected with a box in the grid that entities value was sent to the corresponding input, giving the agent some rudimentary vision and hopefully making it better for more general use.

Thesis

Recreate human behaviour with artificial intelligence in a 2D top down wave shooter game

Two different AI were made to play a game with as humanlike behaviour as possible. One AI made with Behaviour trees and one with a LSTM neural network trained with reinforcement learning. When they were ready testers played with both AI one after another, except that the testers were told that one of the agents was an AI and one was a human. The tester was not told which were which and they had to decide which they thought were the AI. The goal of this was to find out which of the two used techniques were best at imitating human player behaviour.
